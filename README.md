- 🎯 조합 예측 시뮬레이터

- 🧵 소재 분석

## 🎯 주요 기능 설명

### 1. **조합 예측 시뮬레이터** ⭐
```
입력: 성별 + 아이템 + 소재 + 핏 + 기장
출력: 예상 판매량/금액 + 신뢰도
```

**예측 로직:**
- 완전 일치 데이터 → 신뢰도 95%
- 4개 일치 데이터 → 신뢰도 75%
- 3개 일치 데이터 → 신뢰도 60%
- 2개 일치 데이터 → 신뢰도 40%

### 2. **자동 품번 파싱**
```
TWSK10954 → 
- 브랜드: 세르지오타키니
- 성별: 여성
- 아이템: 스커트
- 카테고리: 하의
- 연도: 2025
- 시즌: 가을
```

### 3. **데이터 입력 2가지 방법**
- 📝 수동 입력 (한 건씩)
- 📂 Excel 업로드 (대량)

### 4. **분석 기능**
- 성별/카테고리/아이템별 판매 분석
- 시즌/연도별 트렌드
- 컬러별 선호도
- 조합별 성과 랭킹 (Best/Worst)
- 소재별 성과 분석

## 💡 사용 시나리오

### **Day 1: 초기 설정**
1. 앱 배포
2. Excel 템플릿 다운로드
3. 과거 데이터 10-20건 입력 (테스트)

### **Week 1: 데이터 축적**
1. 팀원들에게 앱 링크 공유
2. 매일 판매 데이터 입력
3. 소재 마스터 데이터 구축
4. 목표: 50-100건

### **Week 2: 분석 활용**
1. 조합 예측 시뮬레이터 사용
2. 성과 랭킹으로 인사이트 도출
3. 소재별 성과 확인
4. 신규 기획 시 예측 참고

### **Week 3-4: AI 준비**
1. 100건+ 데이터 축적
2. AI 모델 추가 요청 (저에게)
3. 예측 정확도 향상
